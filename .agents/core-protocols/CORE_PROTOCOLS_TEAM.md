## About core-protocols team

The core-protocols team is responsible for a set of areas in the project. The areas owned by the core-protocols team correspond to the following GitHub labels:

- `area/adapter/fuse`
- `area/adapter/java-cli`
- `area/adapter/jee`
- `area/adapter/jee-saml`
- `area/adapter/spring`
- `area/adapter/javascript`
- `area/oidc`
- `area/oid4vc`
- `area/saml`
- `area/token-exchange`

The GitHub label `team/core-protocols` is used to label issues, PRs, and anything else associated with our team.

Tasks and issues linked to the core-protocols team areas are high-priority, but you should also be prepared to work on areas other than those.
