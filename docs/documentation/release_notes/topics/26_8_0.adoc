= Security and Standards

== SAML `login_hint` query parameter support

{project_name} can now forward the `login_hint` query parameter to SAML identity providers, allowing destination providers to pre-fill their login form. This behavior is controlled by a new *Use login_hint* option in the SAML identity provider configuration.
