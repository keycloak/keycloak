This {section} provides an overview of the platforms, versions, and configurations that are tested and supported for running {project_name} in production environments.

== Java Virtual Machine

{project_name} requires a Java Virtual Machine (JVM) to run. The following Java Development Kit (JDK) versions are supported:

* OpenJDK 17 (LTS)
* OpenJDK 21 (LTS)
* OpenJDK 25 (LTS)

NOTE: To benefit from the latest performance improvements, we suggest using the latest supported JDK version for production deployments.
This is currently OpenJDK {jdk_version_latest}.

WARNING: Due to OpenJDK limitations, FIPS mode is only supported with OpenJDK 17 and 21 in {project_name}. Consider this when choosing the Java version for your production deployment.

=== Container Image

{project_name} provides official container image that is using OpenJDK {jdk_version_container}. Consider this when writing
custom extensions. For more details on the container image, see <@links.server id="containers"/>.

== Runtime Platform

{project_name} can be deployed in various environments:

* **Bare Metal / Virtual Machines** - Direct installation on Linux or Windows servers
* **Containers** - Docker, Podman
* **Kubernetes** - OpenShift 4.18-4.21, corresponding to Kubernetes API 1.31-1.34

NOTE: Other non-OpenShift Kubernetes distributions (like Amazon EKS or Azure Kubernetes Service etc.) are supported only in a best-effort manner, i.e. support will be provided for issues that
are specific to the supported Kubernetes API or OpenShift versions, but issues that are reproducible only on a particular Kubernetes distribution may not be supported.

NOTE: The supported versions of each respective environment are limited to the versions that are currently supported by the respective vendors. For example, for OpenShift, only the versions that are currently supported by Red Hat are supported by {project_name}.

== Supported Browsers

The {project_name} UI including login pages, Admin Console and Account Console are supported with the following browsers:

[%autowidth]
|===
|Browser | Supported Versions

|Google Chrome | Latest stable version
|Mozilla Firefox | Latest stable version
|Microsoft Edge | Latest stable version
|Apple Safari | Latest stable version
|===

NOTE: Both desktop and mobile versions of the browsers are supported.

NOTE: The Admin Console and Account Console require JavaScript to be enabled. Browsers must support modern web standards including ES6+, WebAuthn, and Web Crypto API for full functionality.
